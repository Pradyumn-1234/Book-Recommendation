# Book-Recommendation-System
A Book Recommendation System that suggests books based on user preferences using Machine Learning and Streamlit

🔍 Overview

This project is a machine learning-based web application that helps users discover books similar to their interests. By entering a book name, the system analyzes its content and recommends other books with similar characteristics. The application provides an interactive and user-friendly interface built using Streamlit.

⚙️ How It Works

The system uses Natural Language Processing techniques to understand and compare books. It combines the title and description of books and converts them into numerical vectors using TF-IDF (Term Frequency–Inverse Document Frequency). Then, it calculates similarity between books using cosine similarity. Based on this similarity, the system recommends the most relevant books.

📊 Features
Search books by title
Get top recommended similar books
Display book details such as author, rating, year, and description
Data visualization using charts and heatmaps
Clean and interactive user interface
🧠 Technologies Used
Python
Pandas and NumPy for data processing
Scikit-learn for machine learning
Matplotlib and Seaborn for visualization
Streamlit for web application development
🎯 Output

The system provides a list of recommended books along with:

Book title
Author name
Rating
Publication year
Short description

This project demonstrates the practical implementation of machine learning and recommendation systems in real-world applications.
